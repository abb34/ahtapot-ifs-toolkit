Ahtapot — ERP Toolkit

Chrome Extension Privacy Policy

This extension does not collect, transmit, or share any personal data. All data stays in your browser.

What data does the extension access?

Ahtapot — ERP Toolkit intercepts network requests made by your browser to ERP pages you visit. This data includes:

  • OData API responses from your ERP instance (e.g. purchase orders, inventory records)
  • The URL of the active ERP page

This data is used solely to display information inside the extension popup and to generate Excel/PDF reports on your device.

Data storage

All captured data is stored locally using Chrome's built-in storage API (chrome.storage.local). This storage is:

  • Local to your browser — never sent to any external server
  • Accessible only to this extension
  • Cleared when you use the "Clear Cache" or "Delete All Templates" functions, or when you uninstall the extension

Data transmission

Ahtapot does not:

  • Send any data to external servers or third parties
  • Use analytics, tracking, or advertising services
  • Collect personally identifiable information
  • Communicate with any server other than your own ERP instance

Permissions used

  • activeTab / tabs — To detect which ERP page is open and link captured data to the correct tab
  • storage — To save templates, environment labels, and sticky notes locally
  • Host permissions (*://*/*) — Required because cloud ERP systems can be hosted on any domain (cloud, on-premise, custom URLs)

Contact

If you have any questions about this privacy policy, please open an issue on the project repository or contact the developer directly.
